# English Version

## üéØ General Concept

LifeHub is a unified system for holistic personal development.  
The MVP aims to combine tasks, habits, addiction tracking, goals, and journaling into one seamless experience.

The user flow:  
Realizing the desire to improve ‚Üí setting goals ‚Üí breaking them down into actionable steps ‚Üí tracking ‚Üí AI-assisted reflection and support.

---

## üë§ Core User Story

1. **The user realizes they want to improve themselves.**
2. **They set one or several goals** ‚Äî manually or with AI assistance.  
   Example: *‚ÄúI want a healthy body.‚Äù*
3. **Each goal is decomposed** into clear, achievable stages.  
   Examples:  
   - ‚ÄúQuit smoking‚Äù  
   - ‚ÄúLose 10 kg‚Äù  
   - ‚ÄúGo to the gym regularly‚Äù
4. **Stages must be achievable** and may be completed in parallel.
5. For each stage, the user creates **tasks / habits / dependencies**, e.g.:  
   - Task: ‚ÄúBuy a gym membership‚Äù  
   - Task: ‚ÄúChoose a calorie-tracking app‚Äù  
   - Dependency: ‚ÄúSmoking‚Äù  
   - Habit: ‚ÄúLog calories daily‚Äù  
   - Dependency: ‚ÄúSugar‚Äù
6. **Tasks, habits, and dependencies can also exist independently of goals.**
7. **Journal** ‚Äî the user writes daily notes:  
   - general reflections  
   - goal-specific reflections  
   AI structures entries and generates a readable summary.
8. **AI assistant analyzes user behavior:**  
   - Reminds about deadlines  
   - Asks follow-up questions if tasks remain unfinished  
   - Detects blockers  
   - Suggests new tasks  
   - Edits or removes outdated tasks  
   - Encourages, supports, explains setbacks  
   - Generates summaries of progress and diary entries

---

## üß© Unified Entity: task / habit / dependency

They share a base model but behave differently:

### **1. Task**
- One-time or recurring.
- May have a deadline.
- Included in daily analysis and summaries.

### **2. Habit**
- Performed daily or on a schedule.
- Has a final duration (e.g., 28 days).
- May allow skips without losing progress.

### **3. Dependency**
- Aimed at eliminating harmful habits.
- Infinite duration.
- **Any relapse resets progress to zero.**
